Создание личных страниц на GitHub Pages

Что такое GitHub и GitHub Pages

GitHub — это платформа для совместной разработки, хранения кода и версионного контроля. Используя GitHub Pages, можно создать сайт для демонстрации своего портфолио или проекта, доступный по адресу username.github.io.

Преимущества личных страниц на github.io

  • Бесплатный хостинг для сайта.
  • Простой способ делиться проектами и создавать портфолио.
  • Поддержка пользовательского домена и HTTPS.

Как создать личную страницу на GitHub Pages

  1. Зарегистрируйтесь или войдите на GitHub.
  2. Создайте новый репозиторий с именем username.github.io, где username — ваш GitHub-логин.
  3. Добавьте файл index.html с содержимым вашего сайта и загрузите его в репозиторий.
  4. Зайдите в настройки репозитория, активируйте GitHub Pages и выберите основную ветку для публикации сайта.
  5. Через несколько минут сайт будет доступен по адресу https://username.github.io.
